# Credit Card Fraud Detection Analysis

## Project Overview
Analyzed 284,807 real credit card transactions to identify 
fraud patterns using Excel, SQL and Power BI.

## Tools Used
- SQL (SQLite) — Data querying and analysis
- Microsoft Excel — Data exploration and visualization
- Power BI — Interactive dashboard

## Dataset
- Source: Kaggle
- Link: https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud
- Size: 284,807 transactions
- Fraud cases: 492 (0.17%)

## Key Insights
1. Only 0.17% of transactions are fraudulent
2. 62% of frauds occur below €50
3. Average fraud amount (€122) higher than genuine (€88)
4. Fraud peaks at late night — hour 22
5. Total money lost to fraud — €60,127.97
6. Highest fraud transaction just under €1000
